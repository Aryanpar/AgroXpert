## 🌟 Overview

**AgroXpert Plus** is an advanced smart agriculture system that integrates **Artificial Intelligence, IoT, and Mobile Technology** to improve farming efficiency.

It helps farmers:

* Detect plant diseases using AI 📸
* Automate irrigation and fertilization 💧
* Monitor real-time environmental data 🌦️
* Get intelligent farming assistance 🤖

---

## 🚀 Key Features

### 🧠 AI & Machine Learning

* Plant disease detection using TensorFlow Lite
* Multi-crop disease support
* Offline AI predictions
* Optimized lightweight model

### ☁️ Weather & Environment

* Live weather updates
* Rain prediction insights
* Temperature & humidity monitoring

### 🤖 AI Chatbot

* Agriculture-focused chatbot
* Answers queries on crops, soil, fertilizers

### 🔌 IoT & ESP32 Automation

* Control **Water Pump & Fertilizer Motor**
* UV light automation
* Soil moisture monitoring
* Real-time sensor data
* Arduino + ESP32 integration
* Solar compatibility

### 📱 Mobile Application (Flutter)

* Disease detection interface
* Smart chatbot UI
* Weather dashboard
* Sensor monitoring system
* Motor control panel
* Login & authentication system
